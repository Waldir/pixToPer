Pixels to Percentages tool

Welcome! I've created this tool in order to make it easier to possition elements inside a responsive container by turning pixels into percentages for the top, left, width, height, and bottom properties.

Pasting a string of numbers

You can paste a four number comma separated string into the first box of the "Element dimensions" and the numbers will be distributed to all the boxes. i.e: 50, 50, 358, 486
This can be very useful if you use scripts in order to quickly extract these properties from programs such as Adobe Illustrator.

Include

The include checkboxes allow you to add or subtract element properties to the css code.
Keep in mind that for positioning, in most cases you will only need a y and x combination of properties. For example top + left will work just as well as bottom + right. However, including a bottom property when you already have a top property can be redundant.
An instance where you can exclude a height property is when the element will have an image inside it with 100% width and will automatically push down the element to match the image's aspect ratio.

Decimal places

You can change the amount of decimal places for the css percentages but keep in mind that more decimals equals more accuracy. Also keep in mind browser support as some browsers truncate decimals.

The container

The container in this tool is not of actual size but it does keep the correct aspect ratio in order to give an acurate visual representation.

The HTML

<div id="container">
  <div id="element"></div>
</div>

Copy to clipboard

Clicking on the icon will copy all the generated css into your clipboard.
